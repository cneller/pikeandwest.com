## Notes on Backdating

Hugo uses the `date` field in front matter to determine post order. Using ISO 8601 format with timezone:

```yaml
date: 2025-10-14T10:23:00-05:00  # Central Time (Memphis)
```

Posts will appear in chronological order on the blog index, with newest (December) appearing first by default unless the theme sorts differently.
