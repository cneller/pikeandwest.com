# AI Prompt: Social Media Captions

> Use this prompt to generate platform-specific social captions from blog content.

## Multi-Platform Prompt Template

```text
You are a social media manager for Pike & West, an intimate art gallery and event venue in Germantown, Tennessee (Memphis area). Generate social media captions to promote a blog post across multiple platforms.

## Brand Voice for Social
- Warm and inviting
- Sophisticated but approachable
- Celebratory and positive
- Helpful without being salesy
- Local pride in Memphis/Germantown

## Blog Post Details

**Title:** [Insert blog title]

**URL:** [Insert full URL]

**Primary Topic:** [Brief description]

**Key Takeaways:**
1. [Takeaway 1]
2. [Takeaway 2]
3. [Takeaway 3]

**Target Persona:** [birthday / anniversary / prom / corporate / graduation / baby-shower]

**Best Visual Element:** [Describe the featured image or suggest visual approach]

## Generate Captions For:

### 1. Instagram Feed Post
Requirements:
- Hook in first line (appears before "...more")
- 150-200 words total
- Include 3-5 relevant hashtags at end
- Add a clear CTA
- Use line breaks for readability
- No emojis (per brand guidelines)

### 2. Instagram Story/Reel Caption
Requirements:
- Very short (under 50 words)
- Focus on one key insight
- Include swipe-up or link in bio CTA
- Designed for quick consumption

### 3. Facebook Post
Requirements:
- 80-150 words
- More conversational tone
- Include link directly in post
- Can ask a question to encourage engagement
- No hashtags (or 1-2 maximum)

### 4. Google Business Profile Post
Requirements:
- Under 100 words
- Focus on local relevance
- Include clear CTA button text
- Professional but warm tone
- Mention Germantown or Memphis

Provide all four versions in clean, copy-paste ready format.
```

## Content Repurposing Prompt

```text
I have this blog post content. Help me create a social media content series from it.

**Blog Post:** [Paste full blog content or key sections]

Generate:

1. **Carousel Post (Instagram)** - Break the content into 8-10 slides
   - Slide 1: Hook/title
   - Slides 2-9: Key points (one per slide)
   - Slide 10: CTA

2. **Quote Graphics** - Pull 3-4 quotable lines that would work as standalone graphics

3. **Question Posts** - Create 3 engagement questions derived from the content

4. **Tips Thread** - Format as a numbered tips list (works for Twitter/Threads)

Keep Pike & West's warm, sophisticated brand voice throughout.
```

## Caption Variation Prompt

```text
I need 3 variations of this social caption for A/B testing:

**Original Caption:** [Paste caption]

**Platform:** [Instagram / Facebook / GBP]

Create 3 variations:
1. **Hook Variation** - Same content, different opening line
2. **CTA Variation** - Same content, different call-to-action
3. **Angle Variation** - Same topic, different emotional approach (e.g., aspirational vs. practical)

Keep all versions on-brand for Pike & West.
```

## Hashtag Research Prompt

```text
Suggest hashtags for Pike & West social posts about [topic].

Requirements:
- Mix of reach sizes (some large, some niche)
- Relevant to Memphis/Germantown area
- Relevant to event/venue industry
- Relevant to the specific event type

Provide:
- 5 high-volume hashtags (100k+ posts)
- 5 medium-volume hashtags (10k-100k posts)
- 5 niche hashtags (under 10k posts)
- 3 local hashtags

Explain which combination works best for different goals (reach vs. engagement).
```
