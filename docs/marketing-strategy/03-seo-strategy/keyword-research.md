# Keyword Research - Pike & West

> Strategic keyword targeting organized by search intent tier and mapped to target personas.

## Keyword Strategy Overview

Our keyword strategy follows a three-tier approach based on search intent:

1. **Tier 1** - High-intent local keywords (ready to book)
2. **Tier 2** - Consideration keywords (comparing options)
3. **Tier 3** - Awareness/educational keywords (planning stages)

---

## Implementation Priority

### Phase 1: Foundation (Month 1-2)

1. Optimize homepage for "event venue Germantown" + "event space Memphis"
2. Create wedding landing page targeting Tier 1 wedding keywords
3. Complete Google Business Profile optimization
4. Implement schema markup for LocalBusiness + EventVenue

### Phase 2: Event Type Expansion (Month 3-4)

1. Create corporate events landing page
2. Create private events landing page
3. Target Tier 1 keywords for each event type
4. Begin Tier 2 blog content

### Phase 3: Content Marketing (Month 5+)

1. Launch Tier 3 educational content calendar
2. Build topic clusters with internal linking
3. Guest post outreach for backlinks
4. Local content partnerships
